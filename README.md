# Employee Management System

A console-based Employee Management System developed using Java and JDBC with Oracle Database.

## Features
- Add Employee
- Update Employee
- Delete Employee
- View Employee by ID
- View All Employees

## Technologies Used
- Java
- JDBC
- Oracle Database
- Eclipse IDE

## Project Structure
- Bean Layer (Employee.java)
- DAO Layer (EmployeeDAO.java)
- Service Layer (EmployeeService.java)
- Utility Layer (DBUtil.java)
- Main Layer (TestEmployee.java)

## Database
Table: employee2
- empid
- empname
- designation
